But as soon as someone portals, how can we be able to interpret those votes?

Obviously it changes the interpretation slightly, but I don't see how it's fundamentally any different.

If someone had portaled lymric A to world C before the game started, it would be similarly noteworthy that trient voted for lymric B in thread B rather than lymric A/C in thread C.  It would become a lot more difficult to determine anyone's preferences between lymric's A and C, because all you can look at is votes before they were conjoined or in the future after they're separated again.

I think it helps to think of the voter as a whole person, and the people voted for as iterations of people.  So I think trient as a whole is up to something, but I'm trying to figure out whether her vote is to kill lymric B, save Dieter B, or save lymric C so I can figure out where she's up to something and what her ultimate goals are.

Portaling doesn't change the voters, they're still whole people with the same goals as before.  It just changes the set of options each voter can vote for on a given day.  So it's interpreting the same person in situations that vary more than usual from day to day.
